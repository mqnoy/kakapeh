### Installing

download the file
[kakapehfile](https://github.com/mqnoy/kakapeh/archive/master.zipl) 


or clone the repository

```
git clone https://github.com/mqnoy/kakapeh.git
```


import database

[kakapeh.sql](https://github.com/mqnoy/kakapeh/blob/master/databases/kakapeh.sql) in databases folder
```
mysql -u yourusername -p kakapeh < kakapeh.sql
```

import project 

```
locate to folder kakapeh ,with NetBeans
```
